# Week 2 Notebooks Overview

This directory contains two core Jupyter notebooks designed to transition students smoothly from basic file handling to automated data retrieval, while demystifying the underlying Python syntax. The flow employs a "data-first" approach, getting students to their "aha!" moment quickly before breaking down the exact mechanics.

*   `1-notebook-code-basics.ipynb` **(Working with Files, Pandas, and Reading Data)**
    *   **Focus:** Getting oriented, loading local data, and pulling data dynamically from the web.
    *   **Key Concepts:** File paths, assigning variables, bringing in the `pandas` library, reading CSVs locally and from GitHub URLs, interacting with APIs (World Bank), and a "just-in-time" introduction to custom functions (`def`) and lists.
    *   **Goal:** Students first learn how to locate themselves and successfully import their first local dataframe. Then, they scale up to demystify where data comes from on the web, fetching information programmatically and leveraging custom-built functions to interact with APIs.

*   `2-python-fundamentals.ipynb` **(Looking Under the Hood: The Grammar of Python)**
    *   **Focus:** Deconstructing the code used in previous notebooks.
    *   **Key Concepts:** Deep dive into data types (strings, integers, floats) and structure mapping (dictionaries).
    *   **Goal:** Returning to the fundamentals of syntax now that students have achieved results. This notebook connects the dots, thoroughly explaining *why* the objects they used in Notebook 1 behaved the way they did.
